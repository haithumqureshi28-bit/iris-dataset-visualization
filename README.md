# Iris Dataset Visualization

## Objective
Understand how to read, summarize, and visualize the Iris dataset using Python.

## Approach
- Loaded the Iris dataset using Seaborn.
- Inspected the dataset using shape, columns, and head().
- Created scatter plot, histogram, and box plot.
- Analyzed relationships, distributions, and outliers.

## Results and Insights
- Petal length differs significantly among species.
- Sepal length and petal length show a positive relationship.
- Histograms reveal data distribution.
- Box plots help identify spread and potential outliers.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
